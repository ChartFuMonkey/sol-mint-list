# Solana Mint Lint

Command line tool written in Typescript to retrieve mint tokens with metadata from the Solana blockchain.

# Usage

This project uses `yarn` and `typescript`.

**Run Locally**

_This uses `ts-node` to run directly from `typescript`._

```
    $ yarn retrieve
```

**Build**

```
    $ yarn build
```
